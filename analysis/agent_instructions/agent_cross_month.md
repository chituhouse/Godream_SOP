# 跨月关联分析 Agent 指令

## 你的任务
汇总 5 个月度分析的结果，做跨月关联分析。

## 分析重点

### 1. 问题生命周期完整追踪
- 哪些问题跨越多个月？
- 问题的演进路径是什么？
- 最终是解决了还是放弃了？

### 2. 用户成长曲线
- 提问方式如何演进？（从模糊到精准）
- 知识盲区如何缩小？
- 解决问题的效率如何变化？

### 3. 工具使用演进
- 从 Codex 到 Claude 的切换发生在什么时候？为什么？
- Serena 的活跃期是什么时候？解决了什么问题？
- 工具选择的经验教训是什么？

### 4. 技术债务演进
- 巨石文件是什么时候开始失控的？
- 硬编码是什么时候开始积累的？
- 重构是什么时候发生的？触发条件是什么？

### 5. 关键转折点识别
- 项目的关键转折点有哪些？
- 每个转折点的触发因素是什么？
- 转折点前后的变化是什么？

## 输入
读取 analysis/overnight/ 目录下的月度分析结果：
- month_2025_09.json
- month_2025_10.json
- month_2025_11.json
- month_2025_12_a.json
- month_2025_12_b.json

## 输出格式
生成一份完整的跨月分析报告：
- analysis/overnight/cross_month.json（结构化数据）
- analysis/overnight/full_index.md（可读汇总）

## 注意事项
- 关注跨月的关联和演进
- 用时间线串起所有发现
- 用中文输出所有分析内容
