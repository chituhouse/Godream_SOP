## 技术决策记录

### 1. 为什么从 CSV 迁移到 SQLite?

**问题**: CSV 在数据量增大时性能下降，查询困难
**决策**: 迁移到 SQLite
**结果**: 查询性能提升 10x，支持复杂查询

### 2. 为什么从 PyQt5 迁移到 PySide6?

**问题**: PyQt5 许可证限制，部分功能老旧
**决策**: 完整迁移到 PySide6
**结果**: 更好的 Qt6 支持，更现代的 API

### 3. 为什么引入 Token 化主题系统?

**问题**: 硬编码颜色值散落在代码各处，主题切换困难
**决策**: 17个Phase逐步清理，引入Token系统
**结果**: 主题切换一键完成，维护成本大幅降低

### 4. 为什么选择相对路径存储?

**问题**: 绝对路径导致项目迁移困难
**决策**: 统一使用相对路径
**结果**: 项目可在不同设备间迁移，支持 NAS 共享
