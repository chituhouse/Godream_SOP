# Godream (鸽梦) 营销宣发系列计划

## 1. 整体策略分析 (基于会议精神)
会议记录明确了一个核心战略方向：**将叙事重心从单纯的“生成工具”转移到“资产管理”。**

### 核心战略支柱：
1.  **信任背书**：利用 **火山引擎 (Volcengine)** 的官方合作伙伴关系（井鸽 x 火山）。
2.  **痛点切入**：AIGC 资产价值巨大但管理混乱。企业/创作者需要的是一个“管家”，而不仅是“生成器”。
3.  **差异化竞争**：我们不仅能生成，核心优势在于“存储 + 复用”。
4.  **商业模式**：低门槛利用“火山渠道价”（约几千元）获得私有化资产管理系统 + 顶级的火山引擎模型能力。
